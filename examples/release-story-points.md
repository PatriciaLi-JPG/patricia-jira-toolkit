# Example: Release Story Points

Prompt:

```text
release story points 26.3.10
```

Expected output shape:

- Release: `26.3.10`
- Scope / JQL used:
- Total issue count:
- Total SP:
- Delivery SP:
- Embed bug count:
- Bug ratio:
- Active issue list:
- Missing SP / DEV / QA estimate items:
- Next actions by owner / ticket:

Notes:

- If a team alias is provided, restrict the query to that team.
- If no team alias is provided, use the configured default team mapping.
- Always include the exact JQL used so another SM can verify the data.
